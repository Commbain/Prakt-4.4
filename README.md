# Life Simulator - Консольный симулятор экосистемы

## Описание
Проект симулирует взаимодействие организмов в экосистеме по принципу "хищник-жертва" с базовым обменом энергией.

## Как запустить

### Требования
- Python 3.9 или выше

```bash
# Клонирование репозитория
git clone https://github.com/Commbain/Prank-4.4.git

# Запуск симуляции
